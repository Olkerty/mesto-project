Проект - Место
Место это учебный проект, состоящий из краткого профиля, всплывающго окна и сетки карточек с интересными местами и их названиями. Имеется возможность
редактировать профиль, добавлять и удалять карточки, а также ставить и снимать "Лайки" и рассматривать фотографии крупным планом. 
При создании проекта были использованы html и css для непосредственной вёрстки javascript для 
, git для контроля версий и figmа для работы с дизайнерским макетом.
