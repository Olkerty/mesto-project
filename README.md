#Проект - Место
Место это учебный проект, состоящий из краткого профиля, всплывающго окна и сетки карточек с интересными местами и их названиями, с встроенной 
валидацией. Имеется возможность редактировать профиль, добавлять и удалять карточки, а также ставить и снимать "Лайки"
и рассматривать фотографии крупным планом. При создании проекта были использованы html и css для непосредственной вёрстки javascript для 
создания скриптов, git для контроля версий и figmа для работы с дизайнерским макетом. Также был использован webpack для сборки проекта.
