Проект - Место
Место это учебный проект, состоящий из краткого профиля, всплывающго окна и сетки карточек с интересными местами и ихназваниями. При создании проекта были использованы html и css для непосредственной вёрстки, git для контроля версий и figmа для работы с дизайнерским макетом. 
GitHub - https://github.com/Olkerty/mesto-project
